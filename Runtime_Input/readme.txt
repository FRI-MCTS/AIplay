Input file should be formatted as following:

First line contains the number of all samples in the file (number of rows - 1) and number of atributes.
All samples must have same format, same number of atributes.

Example:
N C
sample1_att1	sample1_att2	... 	#sample1_attC	
sample2_att1	sample2_att2	... 	#sample2_attC
.
.
.
sampleN_att1	sampleN_att2	... 	#sampleN_attC
